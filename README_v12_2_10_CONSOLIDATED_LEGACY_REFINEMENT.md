# v12.2.10 Consolidated Legacy Refinement

## 반영한 제안

1. 모바일홈과 최신 리포트를 통합 홈 구조로 정리했습니다.
2. 추천 TOP15와 진입 시나리오를 `추천 TOP15 + 진입`으로 합쳤습니다.
3. 전략 추천/검증은 핵심 메뉴에서 뒤로 빼고 참고 자료로 낮췄습니다.
4. 원본 모바일 대시보드 메뉴는 제거했습니다.
5. 기존 보유종목 판단 메뉴는 제거했습니다.
6. 보유주식 관련 뉴스 링크는 추가 API 호출 없이 기존 네이버뉴스 결과를 재활용하므로 비용 영향은 거의 없습니다. 태웅/태웅식품 같은 오매칭 필터를 강화했습니다.
7. TOP15와 보유주식에 2~3줄 기본 정보 설명을 추가했습니다.
8. `네이버뉴스 상세` 명칭을 `주요 뉴스 요약`으로 변경하고 상단 요약 데이터를 생성합니다.
9. ETF는 AI/소프트웨어로 뭉뚱그리지 않고 `ETF · 미국 S&P500`, `ETF · 2차전지`, `ETF · 커버드콜/인컴`처럼 분류합니다.
10. 엑셀 다운로드는 다운로드 센터 하나로 통합했습니다.
11. `/latest/`와 `/mobile/`은 같은 통합 홈 HTML을 사용합니다.
12. 전체 추천 명단은 데이터가 많은 열을 앞으로, TOSS/토스 관련 빈 열을 뒤로 보내고 열폭을 개선했습니다.

## 적용 방법

1. 압축을 풉니다.
2. `.github`, `automation` 폴더를 PC의 `stock-report` 폴더에 그대로 덮어씁니다.
3. GitHub Desktop에서 변경사항 확인
4. Summary: `apply v12.2.10 consolidated legacy refinement`
5. Commit to main
6. Push origin
7. Actions에서 Run workflow 실행
